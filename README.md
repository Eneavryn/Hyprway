# Hyprway
# My personal hyprland workflow and confs
## I basically used hyprland with a bunch of utilities along with noctalia-shell to get running my own simpler hyprland setup
### This repo will be including my hyprland.lua config file because it has bunch of modification and snippets for noctalia compatibility like how to set custom theme for launchers, how to enable blur for noctalia given simply toggling the blur toggle doesn't works, a custom noctalia color scheme and way to add it to noctalia, config file for greetd to use tuigreet and hyprland, custom alacritty toml config in case noctalia template doesn't works for alacritty.

- For greeter I'm using greetd with greetd-tuigreet
- for shell it's noctalia-shell and not the latest noctalia git which is in alpha release where stability is not guranteed.
- no need to install quickshell for noctalia-shell but noctalia uses and comes with a custom fork of quickshell called noctalia-qs.
